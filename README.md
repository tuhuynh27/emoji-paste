# Emoji Paste

Visit site: [emoji.keva.dev](https://emoji.keva.dev)

- Quick find emojis
- Click to copy
- Keyboard-only mode
- Recent used emojis
- Update emoji list (in coming)

**For Firefox:** Visit `about:config` and set `dom.events.asyncClipboard.clipboardItem` to `true` to enable `ClipboardItem`.
